## 📥 نصب و اجرا

```bash
# کلون کردن پروژه
git clone https://github.com/nikzad-avasam/downloader.git
cd downloader

# نصب پیش‌نیاز
pip install requests

# اجرا
python github_downloader.py
🎯 چطور استفاده کنم؟
1️⃣ برنامه رو اجرا کن
bash
python github_downloader.py
2️⃣ توکن بده (اختیاری)
از اینجا توکن بساز تا محدودیت برداشته بشه

3️⃣ حالت رو انتخاب کن
دانلود مستقیم: فایل‌ها رو دانلود میکنه

استخراج لینک: لینک‌ها رو تو HTML قشنگ ذخیره میکنه

4️⃣ لینک پوشه رو بده
bash
https://github.com/zorochan32/My-downloader/tree/main/downloads
5️⃣ تمام! 🎉
فایل‌ها خودکار دانلود میشن!

✨ ویژگی‌ها
📥 دانلود کل پوشه با حفظ ساختار درختی

🔗 استخراج لینک‌های دانلود مستقیم

📊 تاریخچه HTML با تب‌های مجزا

⚡ سرعت بالا با نوار پیشرفت

🔑 پشتیبانی از توکن (5000 درخواست/ساعت)

📋 مثال استفاده
bash
🔗 لینک پوشه گیت‌هاب: https://github.com/zorochan32/My-downloader/tree/main/downloads
